Installation & Running
----------------------

1. Clone repository:  
`git clone https://github.com/bugs-intern/bug-intern-browser-modules.git`	
2. Install dependencies:  
`cd bug-intern-browser-modules`  
`npm install`  
3. Run tests:  
`npm test`
